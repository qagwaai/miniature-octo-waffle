# Repository Map

This repository is organized as Orion's durable Stellar knowledge base.

## Primary Sections

| Path | Purpose |
| --- | --- |
| [docs/stellar-project-north-star.md](stellar-project-north-star.md) | Canonical project direction and decision filter |

## Repository Conventions

- Prefer short, durable docs over scattered notes.
- Date-stamp snapshots and planning artifacts.
- Separate canonical direction, current state, and planning backlog.
- Keep ownership explicit when a note touches Nova or Forge responsibilities.
